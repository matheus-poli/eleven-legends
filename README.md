# ⚽ Eleven Legends

> Football Manager + Gacha + Stream Integration

## 🧠 Overview

| | |
|---|---|
| **Genre** | Football Manager + Gacha + Stream Integration |
| **Platform** | PC (Steam + Itch.io) |
| **Engine** | Godot 4 |
| **Initial Focus** | Match simulation (no graphical UI) |

## 🗂️ Project Structure

```
eleven-legends/
├── src/              # GDScript source code
│   ├── simulation/   # Match simulation engine
│   ├── gacha/        # Gacha / player card system
│   ├── manager/      # Team & squad management
│   └── stream/       # Stream integration hooks
├── assets/           # Sprites, audio, fonts
├── scenes/           # Godot scenes (.tscn)
├── tests/            # Unit & integration tests
└── docs/             # Design documents & specs
```

## 🚀 Getting Started

1. Install [Godot 4](https://godotengine.org/download)
2. Clone this repository
3. Open `project.godot` in Godot 4
4. Run the project (`F5`)
