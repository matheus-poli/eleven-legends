# Eleven Legends

Football Manager web game built with Next.js, TypeScript, TailwindCSS, and DaisyUI.

## Tech Stack

- **Next.js 15** (App Router) — Framework
- **TailwindCSS 4 + DaisyUI** — Styling
- **anime.js** — Animations
- **dnd-kit** — Drag & Drop
- **Zustand** — State management
- **seedrandom** — Deterministic RNG
- **Vitest** — Testing

## Development

```bash
pnpm install
pnpm dev
```
